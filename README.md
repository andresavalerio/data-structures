## How to use this repository
Remember that **it is not recommended to copy code without trying first.** Watch videos, read books and seak help from more experienced programmers before looking into the source code.

It you are a *student*, you can:

- Get hints on how to create your own projects;
- Seak simple explanations on how each data structure behave;
- Improve your solutions with code that already works;

If you are a *contributor*, please, feel free to send me feedback and add your own pull requestes.

This repository is structures in:
- Source code, in C or C++;
- `README.md` files, explaining how the data structure works and how to run the code;
- Header files if necessary to maintain the repository organized.

## Why C/C++? Why not other language?
Firstly, C and C++ are more challenging languages. Most of the modern languages like Python or JavaScript already have the basic data structures built in, so it's easier to relly on this tools to create projects. Both these languages, specially C language, have less features and make the code more focused on the understanding of the algorithms.

Secondly, these are faster and more efficient languages. They make it easier to run lots of information withing seconds.

Lastly, they are everywhere on Computer Science. From history to our best Operational Systems, learning those two languages can open your vision on the bases of Computer Science.

Remember that you can study the subject on the language you prefer most, but as a personal choice, I believe that these are best languages to work with.
